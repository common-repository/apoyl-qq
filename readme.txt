== Description ==

这是一款实现QQ互联一键登录网站，让用户不在繁琐去注册用户，一键实现QQ登录，极大的方便用户登录网站.

## 插件功能

* 支持登录一键QQ登录
* 支持自动同步QQ昵称到网站用户名,前提满足wp用户的注册要求
* 支持一键QQ登录用户，自定义角色：订阅者，贡献者，作者，编辑，管理员，这个站点没有任何用户角色
* 支持绑定记录管理
* 支持已登录账号可以绑定QQ
* 支持解绑账号
* 支持中文名称
